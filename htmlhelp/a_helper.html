class AHTMLHelper
Pure utility class used to format data into viewable HTML.
Method Description
static void formatBinaryData
(
   const char *pccName,
   AHTML &html,
   char *pData,
   DWORD dwLength = 0,
   WORD wColumns = 32
) public 		Takes a char* buffer of given length and displays it in a nice printable and hex format in a table.
Default column width is 32 characters, length of 0 means use strlen() to determine length.
Output looks similar to (in table form):
   {pccName} Length: {dwLength}
   0000: ABCD.... 65 66 67 68 00 00 00 00

Returns: Nothing
static void dumpBinaryDataAsHex
(
   AHTML &html,
   char *pData,
   DWORD dwLength,
   WORD wColumns = 32
) public 		Takes a char* buffer of given length and dumps it in a printable format without any HTML tags. Default column width is 32 characters.
Output looks similar to:
   0000: ABCD....
Returns: Nothing
static void dumpBinaryDataAsPrintable
(
   AHTML &html,
   char *pData,
   DWORD dwLength,
   WORD wColumns = 32
) public 		Takes a char* buffer of given length and dumps in hex format. Default column width is 32 characters.
Output looks similar to:
   65 66 67 68 00 00 00 00
Returns: Nothing
